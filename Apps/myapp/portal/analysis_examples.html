Quick Instruction

  1. Open /analysis and login.
  2. In Image Upload, select one or more PNG/JPG files.
  3. In CSV Upload, select one CSV file or an entire folder.
  4. Wait for queue status to become completed.
  5. Use Query Result to inspect metrics and export JSON.

Note: current cloud setup can run in CPU mode. It is slower than GPU but fully functional.
